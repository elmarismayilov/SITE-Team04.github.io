Group Project

This project involves developing a website using HTML, CSS, and JavaScript. The website will consist of five pages, each with a distinct purpose and design. The Home page introduces visitors to Team 4 and serves as the starting point of the website. It provides general information about the team, outlining its purpose and objectives. This page sets the foundation for the rest of the site and welcomes visitors with an engaging overview. The Team Members page is designed to present detailed information about each member of Team 4. For every team member, the page includes a photograph, details of their academic background, and personal information. This section aims to provide a well-rounded introduction to the individuals behind the team. The Hour of Code page documents Team 4's previous task, titled "Hour of Code." It features a written explanation of the task's purpose and outcomes, a video showcasing the entire process, and relevant photos that highlight key moments. The page demonstrates the team’s collaborative efforts and accomplishments during this project. The Web Project page provides a detailed overview of the website development process. It includes individual progress reports that showcase each member's contributions, a video that demonstrates all the code used in the project, and an explanation of the website's structure. This page is intended to offer insights into the technical and collaborative aspects of creating the website. The Code-Art page showcases interactive and creative artwork designed by Team 4 using JavaScript. This section highlights the team’s ability to apply programming skills in innovative and visually appealing ways, adding an artistic dimension to the website. The purpose of this website is to highlight Team 4's technical and creative skills, showcase their collaborative work, and present their projects in an organized and engaging format.
